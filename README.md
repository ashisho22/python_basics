PROGRAM NO :1

Odd and Even Number Checker (Python)
This is a simple Python program that takes a number as input from the user and checks whether it is even or odd.

Features :-

* Takes an integer input from the user
* Uses conditional statements (if-else)
* Determines whether the number is even or odd

Logic :-

The program uses the modulus operator (%) to check the remainder when the number is divided by 2.
If the remainder is 0, the number is even; otherwise, it is odd.

Example :-

Enter a number: 8

Output :-
Number is Even

Concepts Used :-

* Input/Output
* Conditional Statements
* Modulus Operator

PROGRAM NO :2

Positive, Negative or Zero Checker (Python)

This is a simple Python program that takes a number as input from the user and determines whether it is positive, negative, or zero.

Features :-

* Takes an integer input from the user
* Uses conditional statements (if-elif-else)
* Identifies whether the number is positive, negative, or zero

Logic :-

The program checks the value of the number using conditional statements.
If the number is greater than 0, it is positive; if it is less than 0, it is negative; otherwise, it is zero.

Example :-

Enter a number: -5

Output :-
Number is Negative

Concepts Used :-

* Input/Output
* Conditional Statements

PROGRAM NO :3

Largest of Two Numbers (Python)

This is a simple Python program that takes two numbers as input from the user and determines which one is larger, or if both are equal.

Features :-

* Takes two integer inputs from the user
* Uses conditional statements (if-elif-else)
* Compares both numbers and identifies the larger one
* Handles the case when both numbers are equal

Logic :-

The program compares both numbers using conditional statements.
If the first number is greater than the second, it is the largest; if the second is greater, it is the largest; otherwise, both numbers are equal.

Example :-

Enter first number: 12
Enter second number: 8

Output :-
12 is larger than 8

Concepts Used :-

* Input/Output
* Conditional Statements
* Comparison Operators
