My Learning Journey & Mistakes

In this repository, I am also sharing the mistakes I made while starting and how I fixed them.

⸻

Mistake 1: No Folder Structure

At the beginning, I did not know that I could create folders and organize my files inside them.
I created separate files for everything without any structure.

Because of this, my repository became unorganized.
All types of programs were mixed together, and it was difficult to find specific files.

Solution

I looked at other users’ repositories to understand how they organize their projects.
Then I searched online and learned how to create folders and manage files properly.

⸻

Mistake 2: Writing Everything in One README

I was writing all program details (loops, conditionals, etc.) in a single main README file.
This made it confusing because it was not clear which program belonged to which topic.

Solution

When I started adding loop programs, I realized that it would be difficult for anyone to find specific programs.
They would have to open multiple folders again and again.

So I decided:

* The main README will only contain my mistakes and learning journey
* Each folder (like conditions, loops, etc.) will have its own README for programs

This made the repository clean and easy to understand.
