Sum of N Numbers (Python)

This is a simple Python program that calculates the sum of the first N natural numbers based on user input.

Features :-

* Takes a number (N) as input from the user
* Uses loop to calculate the sum
* Displays the final sum

Logic :-

The program uses a loop to add numbers from 1 to N.
It keeps adding each number to a running total and finally prints the sum.

Example :-

Enter a number: 5

Output :-
Sum is: 15

Concepts Used :-

* Input/Output
* Loops
* Arithmetic Operators

Reverse a Number (Python)

This is a simple Python program that takes a number as input from the user and reverses it using a while loop.

Features :-

* Takes a number as input from the user
* Uses a while loop to reverse the number
* Extracts digits using modulus (%)
* Displays the reversed number

Logic :-

The program repeatedly extracts the last digit of the number using modulus (%).
It builds the reversed number by multiplying the current result by 10 and adding the extracted digit.
Then it removes the last digit from the original number using integer division (//) until the number becomes 0.

Example :-

Enter a number: 1234

Output :-
Reversed: 4321

Concepts Used :-

* Input/Output
* Loops (while loop)
* Arithmetic Operators


PROGRAM :6

Count Digits in a Number (Python)

This is a simple Python program that takes a number as input from the user and counts the total number of digits in it.

Features :-

* Takes a number as input from the user
* Uses a while loop to count digits
* Displays the total number of digits

Logic :-

The program repeatedly removes the last digit of the number using integer division (//).
Each time a digit is removed, a counter is increased by 1.
This process continues until the number becomes 0, giving the total count of digits.

Example :-

Enter a number: 12345

Output :-
Digits: 5

Concepts Used :-

* Input/Output
* Loops (while loop)
* Arithmetic Operators
