Sum of N Numbers (Python)

This is a simple Python program that calculates the sum of the first N natural numbers based on user input.

Features :-

* Takes a number (N) as input from the user
* Uses loop to calculate the sum
* Displays the final sum

Logic :-

The program uses a loop to add numbers from 1 to N.
It keeps adding each number to a running total and finally prints the sum.

Example :-

Enter a number: 5

Output :-
Sum is: 15

Concepts Used :-

* Input/Output
* Loops
* Arithmetic Operators
